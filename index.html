Weather forecast website using API

I wanted to recreate something close to what we use everyday as forcasting tool. I had a lot of limitations due to the restricted free access OpenWeather gave me. It gives the possibility to search for your city, add/delete it to your favourites (LocalStorage), see today's forecast, 3h predictions and a rain radar.

Consult live site

Custom password generator arcade themed

Generator password are very common in Javascript projects. I wanted to make it more interesting with a special design inspired by retro arcade gaming. I had a lot of fun with css animations and the global design. About his functionalities, it allows to generate a password with type of elments you want : special characters, letters and numbers.

Consult live site

To Do List using LocalStorage

Once again, a very common project I wanted to push as far as I could. This To Do List is using LocalStorage to store your tasks to be done or the finished one. I also did a filter to sort task by their state (to do/done).
